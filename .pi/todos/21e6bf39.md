- Given a RemoteRepo, apply rules:
  1. is_fork → rules.forks
  2. is_archived → rules.archived
  3. default → rules.base
- Result: (repo, expected_path)
- Then determine action:
  - EXISTS_CORRECT: remote URL matches and local path == expected_path → update (stash + pull --rebase)
  - EXISTS_WRONG_PATH: remote URL matches but path != expected_path → propose move
  - NOT_FOUND: no local match → clone
- Output: Vec<Action> where Action is one of Update/Move/Clone
