- Define Rust structs for the config (workspace, rules, sources)
- Parse with serde + toml
- Support `~/.config/gitjuggling/config.toml` default + `--config` override
- CLI arg for `--config`

Config structure:
```toml
default_workspace = "personal"

[workspace.personal]
root = "..."
github_owners = ["vrischmann"]
forgejo_url = "..."      # optional
forgejo_user = "..."     # optional
forgejo_token_cmd = "..." # optional

[workspace.personal.rules]
base = "..."
forks = "..."
archived = "..."
```

Each workspace can optionally have forgejo fields. If missing, no Forgejo sync for that workspace.
