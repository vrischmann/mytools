- Define a common `RemoteRepo` struct: name, owner, is_fork, is_archived, is_mirror, clone_url, source (GitHub/Forgejo)
- GitHub client: use `gh auth token` to get token, then call GitHub REST API (`/users/{owner}/repos` or `/orgs/{org}/repos`) with pagination. Handle multiple owners.
- Forgejo client: execute `op read` with the configured command to get token, then call Forgejo API (`/api/v1/users/{user}/repos`) with pagination. Filter out mirrors (`mirror == true`).
- Both clients return `Vec<RemoteRepo>`
